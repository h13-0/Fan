# Fan
由PD快充驱动的服务器风扇，可调转速，可自选高风速和静音模式。  
供电支持Type-C PD快充协议和DC 12V直流供电。  
转速调节精度为300档，支持一键满速。  
核心主控使用STM8，Release界面有编译完成的二进制文件，无需配置IDE即可刷入。
